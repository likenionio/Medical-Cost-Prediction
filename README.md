# Medical Cost Prediction (個人の医療費予測モデル)
📌 プロジェクト概要 (Overview)
本プロジェクトは、個人の属性データ（年齢、性別、BMI、喫煙の有無、居住地域など）をもとに、将来の医療費を予測する機械学習の回帰モデルを構築したものです。
データの前処理から、複数のアルゴリズムの比較検証、精度評価までの一連のデータサイエンス・プロセスを実装しました。

🛠 使用技術 (Tech Stack)
言語: Python 3

データ処理: pandas, NumPy

機械学習: scikit-learn (Linear Regression, Random Forest 等)

可視化: matplotlib, seaborn

📊 データセット (Dataset)
Kaggleで公開されている [可疑链接已删除] を使用しました。

🚀 実装の工夫点・プロセス (Workflow & Highlights)
探索的データ分析 (EDA): 各特徴量の分布や相関を可視化し、「喫煙の有無」と「BMI」が医療費に強く影響を与える仮説を立てました。

データ前処理 (Preprocessing): カテゴリ変数に対して One-Hot Encoding を適用しました。

モデルの選定と学習 (Modeling): 単一の手法に依存せず、複数の回帰アルゴリズムを実装し比較しました。

モデル評価 (Evaluation): RMSE および R2スコア を用いて、モデルの予測精度を定量的に評価しました。

💡 考察 (Conclusion)
単に精度を追求するだけでなく、「どの特徴量が予測に大きく寄与しているか」を分析することで、ブラックボックスになりがちな機械学習モデルの解釈性を高めるよう意識して実装しました。
